To compile a file, use :

	"make parse file=<filename>" (<filename> does not include the ".c" extension)


To interprete the assembly file generated by the previous command, use :

	"make interprete file=<filename>" (<filename> does not include the ".s" extension)


To rebuild the parser after modifying lex or yacc files, use :

	"make compiler" (or simply "make")


To recompile the interpreter after modifying "interpreter.c", use :

	"make interpreter"
